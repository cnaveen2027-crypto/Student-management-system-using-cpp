# Student-management-system-using-cpp
The Student Management System is a console-based application developed in C++ designed to efficiently manage student records using file handling techniques. This project demonstrates the practical implementation of object-oriented programming concepts along with persistent data storage.


# Student Management System (C++)

A console-based Student Management System built using C++ with file handling.

## Features
- Add Student
- View All Students
- Search Student
- Update Student
- Delete Student
- Persistent Storage using Binary Files

## Technologies Used
- C++
- File Handling (fstream)

## How to Run

```bash
g++ main.cpp -o sms
./sms
